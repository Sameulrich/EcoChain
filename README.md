# EcoChain

A decentralized environmental action tracking and sustainability reward system for incentivizing green behavior on Stacks blockchain.

## Features

- Environmental action tracking with impact-based rewards
- Eco-warrior level progression system with green bonus multipliers
- Eco point accumulation and redemption
- Sustainability initiative optimization with time-based penalties
- Comprehensive environmental program statistics

## Smart Contract Functions

### Public Functions
- `start-eco-action` - Begin environmental action
- `complete-eco-action` - Complete action and earn rewards
- `claim-eco-rewards` - Claim accumulated eco points
- `optimize-sustainability-initiatives` - Optimize initiatives with point investment
- `complete-sustainability-optimization` - Complete optimization process

### Read-Only Functions
- `get-eco-action-count` - Get user's total eco actions
- `get-eco-point-balance` - Get user's eco point balance
- `get-eco-level` - Get user's eco-warrior level
- `get-eco-program-stats` - Get overall program statistics

## Reward System
- Base reward: 20 points per action
- Green bonus: 7 points per level (max level 10)
- Optimization multiplier: 3x for sustainability initiatives

## Usage

Deploy the contract to create an environmental incentive system where eco-warriors can track green actions, earn rewards, and optimize sustainability initiatives.
